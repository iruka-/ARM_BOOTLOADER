ReadMe:

--------------------------------------------------------------
  これはＣＱ出版 2011-05 IF付録基板 FRK-RX62N 用の
  USB-CDCエコーバックサンプルです。
--------------------------------------------------------------
  ルネサスのUSBサンプル・フレームワークを流用しています。
  
--------------------------------------------------------------
  コンパイル環境は、特電さんが配布しているcygwin版 rx-elf-gcc
  もしくは、Linux環境で普通に作成されたrx-elf-gccを用います。
  (KPIT版のrx-elf-gccでも試しては居ませんがコンパイル可能かと
   思われます)
--------------------------------------------------------------
  ビルドにはGNU make を使用します。
--------------------------------------------------------------

注意：
  RenesasのRXコンパイラ用ではありません。 gcc専用です。
--------------------------------------------------------------





試し方：
--------------------------------------------------------------
	src/RX62N-0000.mot を ルネサスのFDT 4.08 BasicというFlash
	書き込みツールにて書き込みます。
	書き込み時は基板上のMD0(JPP1)ジャンパーをクローズして通電
	する必要があります。また、そのときのUSB infファイルは、
	FDT 4.08 Basicに付属のものを使ってインストールします。
--------------------------------------------------------------
動作
--------------------------------------------------------------
    単純に、仮想COMポートに送られてきた文字をエコーバックで
    返します。

    D:> w32term -c＜仮想COMポート番号＞  -t
    を実行すると、エコーバック速度のベンチマークテストを行な
    います。
    
    ＜仮想COMポート番号＞を知りたい場合は、
    D:> w32term -p
    で一覧を取得してください。
